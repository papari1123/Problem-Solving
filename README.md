# Problem-Solving
## 기초
알고리즘 풀이에 필요한 문법 및 기초 알고리즘에 대한 소스코드.

## 문제
이전에 푼 문제와 풀이 방법이 중복되지 않는 문제들을 수록. 비슷한 문제는 수록 가능.

## 응용필수
문제중에서 그 풀이방법이 다양하게 응용될 수 있어 복습이 필요할 것 같은 문제 수록.

## 주의사항
문제 풀이하다가 실수하거나, 시간을 많이 까먹는 부분에 대해 서술.
